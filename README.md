## Bookstore Mobile App
A full-featured React Native (Expo) bookstore application with authentication, cart system, wishlist, checkout, and order history — powered by **Firebase**.

---

### Project Overview
**Bookstore Mobile App** is an e-commerce mobile application that allows users to:
  - Browse books by categories
  - View detailed book information
  - Add books to cart
  - Add books to wishlist ❤️
  - Checkout with stock validation
  - View order history
  - Manage profile information
The app is built using modern mobile development practices and integrates with **Firebase Realtime Database + Firebase Authentication**.

---

## Main Features
### Authentication
  - Email & Password Login
  - Signup with validation
  - Forgot Password
  - Change Password
  - Firebase Authentication integration

### Home
  - Top Books
  - Latest Books
  - upcoming Books
  - Dynamic book loading from Firebase

### Categories
  - Browse books by category
  - Search by
      - Title
      - Author
      - Publisher
      - ISBN

### Book Details
  - Full book information
  - Add to cart
  - Add/Remove from wishlist
  - Real-time stock display

### Cart system
  - increase/Decrease quantity
  - Remove item
  - Real-time total calculation
  - Checkout flow
  - Stock validation before purchase
  - Auto cart clearing

### Account Management
  - Profile display
  - Edit Profile (username, phone, address)
  - Change password
  - Order history
  - Notofications
  - Logoout
